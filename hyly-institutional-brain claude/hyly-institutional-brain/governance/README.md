# Governance

This folder contains the rules for managing the Hyly Institutional Brain repository.

It does **not** contain knowledge. It contains the policies, processes, and standards that govern how knowledge is created, reviewed, versioned, and maintained.

## Contents

| File | Purpose |
|------|---------|
| `OWNERSHIP.md` | Repository owners and domain stewards |
| `CONTRIBUTING.md` | How to contribute to this repository |
| `REVIEW_WORKFLOW.md` | Review and approval process |
| `KNOWLEDGE_LIFECYCLE.md` | How knowledge is created, maintained, and retired |
| `VERSIONING.md` | Versioning standards for knowledge assets |
| `STANDARDS.md` | File naming, structure, and formatting standards |
